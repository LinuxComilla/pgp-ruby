A command line wrapper around the GNU PGP (GPG) library to make it sane.

Requires Ruby 1.9.

## Usage

Friendly human commands to wrap `gpg (1)`'s ugly ones.

```console
$ pgp sign 12345EFA

$ pgp push 12345EFA

$ pgp pull

$ pgp list
```
